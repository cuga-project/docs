🏢 ETE LiteLLM (Recommended for IBM Teams)

The default and recommended configuration for IBM team members.

Setup Instructions

  1. Access Portal

    • Visit ETE LiteLLM Portal
    • Login with your IBM credentials

  2. Generate API Key

    • Navigate to the API keys section
    • Generate a new API key
    • Copy the key (starts with ete-)

  3. Configure Environment

  4. Verify Configuration

Benefits

  • ✅ Free for IBM employees
  • ✅ High performance and reliability
  • ✅ IBM security compliance
  • ✅ Seamless integration

🌐 OpenAI

Popular choice for general use with excellent model quality.

Setup Instructions

  1. Create Account

    • Visit platform.openai.com
    • Sign up or log in to your account

  2. Generate API Key

    • Navigate to API Keys
    • Click "Create new secret key"
    • Copy the key (starts with sk-)

  3. Configure Environment

  4. Verify Configuration

Cost Considerations

  • GPT-4: ~$0.03 per 1K input tokens, ~$0.06 per 1K output tokens
  • GPT-3.5-turbo: ~$0.0015 per 1K input tokens, ~$0.002 per 1K output tokens

🔵 IBM WatsonX

Enterprise-grade AI platform with IBM's latest models.

Setup Instructions

  1. Access WatsonX

    • Visit IBM WatsonX
    • Create a new project or access existing one

  2. Get Credentials

    • Project ID: Found in project settings
    • API Key: Generate in project credentials
    • Region/URL: Your deployment region

  3. Configure Environment

  4. Verify Configuration

Benefits

  • ✅ Enterprise security
  • ✅ IBM's latest AI models
  • ✅ Compliance features
  • ✅ Scalable infrastructure
